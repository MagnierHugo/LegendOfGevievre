# Legend of Gevievre

A 2D survivor-like game created for a Game Jam with the theme **"It's not a bug, it's a feature."**

What happens when standard game mechanics are intentionally broken? In Legend of Gevievre, every weapon and power-up is designed to misbehave in unique and challenging ways. Navigate procedurally generated maps, battle relentless monsters, and master the art of fighting with tools that refuse to work as intended.

## Key Features

- **Procedurally Generated Maps**: Every playthrough offers a fresh challenge with maps created using Wave Function Collapse algorithm
- **3 Distinct Monster Types**: Each enemy has unique AI behaviors that will test your survival skills
- **4 "Bugged" Weapons**: Each weapon comes with 3 upgrade tiers and refuses to work the way it should
